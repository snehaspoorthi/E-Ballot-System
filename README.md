# 📌 E-Ballot-System

## E-Ballot: Secure Digital Electoral Management System

A full-stack web application for **secure and transparent digital voting**, built using modern web technologies.

---

## 🚀 Features

- 🔐 Developed a full-stack application using **Next.js (frontend)** and **Node.js (backend)**  
- 👥 Implemented **role-based authentication** for voters and administrators  
- 🔄 Designed RESTful APIs for **voter registration, candidate management, and vote processing**  
- 🗄️ Integrated **Neon PostgreSQL (serverless database)** for scalable data storage  
- ✅ Enforced **one-person-one-vote validation** to prevent duplicate or fraudulent voting  

### 🧾 Vote Receipt System (Key Feature)
- Generates a **unique receipt code** after successful voting  
- Allows users to **verify their vote status** using the receipt code  
- Enhances **transparency and trust** in the voting process  

- 🎨 Built a responsive UI with **Next.js**  
- 🏗️ Structured using **modular and scalable architecture principles**  
- 🔒 Focused on **security, transparency, and real-world election system simulation**

---

## 🛠️ Tech Stack

- **Frontend:** Next.js  
- **Backend:** Node.js (Express)  
- **Database:** Neon PostgreSQL  
- **Other:** REST APIs, Git
